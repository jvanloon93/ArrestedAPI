This is a simple landing page that demonstrates a web API running on Linux with an Nginx web server, MySql database, and built in C#.NET Core.

API Endpoints

GET /api/Quotation
GET /api/Quotation/Id
GET /api/Quotation/Season/SeasonId
GET /api/Quotation/Season/SeasonId/Episode/EpisodeId
